Projetos reais desenvolvidos para resolver problemas reais

Entregue · Fev – Mar 2026

Controle de Caixa

Sistema web desenvolvido para resolver um problema real na empresa onde trabalho. O fechamento de caixa era feito manualmente com erros frequentes e o controle de cheques era uma planilha desorganizada. Utilizado diariamente pelos funcionários, eliminou os erros de fechamento e centralizou todo o controle financeiro em um único lugar — sem instalação ou servidor.

  • Eliminou erros manuais no fechamento de caixa diário
  • Alertas automáticos de cheques vencendo
  • Autenticação por usuário com histórico de sessões
  • Relatórios formatados para impressão
HTML CSS JavaScript LocalStorage
